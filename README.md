# goit-rdb-fp

Фінальний проєкт з теми **Relational Databases**.

## Тема проєкту
Робота з набором даних `infectious_cases.csv` у MySQL:
- створення схеми `pandemic`
- імпорт даних
- нормалізація таблиці до 3НФ
- аналітичний запит для `Number_rabies`
- побудова атрибутів за допомогою вбудованих SQL-функцій
- створення та використання власної SQL-функції

## Структура репозиторію

- `final_project.sql` — SQL-код фінального проєкту
- `screenshots/` — скриншоти виконання завдань
- `infectious_cases.csv` — вихідний CSV-файл з даними

## Що виконано

### 1. Створення схеми та імпорт даних
- створено схему `pandemic`
- імпортовано таблицю `infectious_cases`
- перевірено кількість завантажених рядків

### 2. Нормалізація до 3НФ
Створено дві нормалізовані таблиці:
- `entities`
- `infectious_cases_normalized`

### 3. Аналіз даних
Для кожної унікальної комбінації `Entity + Code` обчислено:
- середнє значення `Number_rabies`
- мінімальне значення
- максимальне значення
- суму значень

Результат:
- відфільтровано порожні значення
- відсортовано за середнім значенням у порядку спадання
- виведено 10 рядків

### 4. Різниця в роках
Для колонки `year` побудовано:
- дату першого січня відповідного року
- поточну дату
- різницю в роках між цими датами

### 5. Власна функція
Створено функцію:
- `year_difference_by_year(input_year INT)`

Функція приймає рік і повертає різницю в роках між поточною датою та датою виду `YYYY-01-01`.


## Примітка
Імпорт CSV-файлу виконувався через **Table Data Import Wizard** у MySQL Workbench.
Для коректного імпорту:
- `Year` імпортувався як `INT`
- усі інші текстові та числові колонки імпортувалися як `TEXT`
